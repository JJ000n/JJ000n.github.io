## `''.join(iterable)`

문자열 리스트(또는 이터러블)를 하나의 문자열로 합치는 함수.

```python
'구분자'.join(iterable)
```

- `iterable` 안의 원소는 **모두 문자열**이어야 함
- 구분자 없이 합칠 때는 `''` 사용

```python
# 기본 사용
words = ['Hello', 'World']
print(' '.join(words))   # Hello World
print(''.join(words))    # HelloWorld
print(', '.join(words))  # Hello, World

# 리스트 → 문자열 변환 (코테 자주 사용)
chars = ['a', 'b', 'c', 'd']
print(''.join(chars))    # abcd

# 숫자 리스트는 직접 불가 → map으로 변환 후 사용
nums = [1, 2, 3]
print(''.join(map(str, nums)))  # 123
```

## `set` / 부분집합 함수

`set`은 **중복 없는 순서 없는** 자료구조.

```python
a = {1, 2, 3}
b = {1, 2, 3, 4, 5}
```

### 주요 부분집합 연산

| 연산 | 의미 |
|------|------|
| `a.issubset(b)` | a가 b의 부분집합인지 (a ⊆ b) |
| `a.issuperset(b)` | a가 b의 상위집합인지 (a ⊇ b) |
| `a <= b` | `issubset`과 동일 |
| `a < b` | **진부분집합** (a ⊂ b, a ≠ b) |
| `a >= b` | `issuperset`과 동일 |

```python
a = {1, 2, 3}
b = {1, 2, 3, 4, 5}

print(a.issubset(b))    # True  → a ⊆ b
print(a.issuperset(b))  # False → a ⊉ b
print(a <= b)           # True
print(a < b)            # True  → 진부분집합 (a ≠ b)
print(a < a)            # False → 같으면 진부분집합 아님
```

### 집합 연산 정리

```python
a = {1, 2, 3}
b = {3, 4, 5}

print(a | b)   # {1, 2, 3, 4, 5}  합집합
print(a & b)   # {3}              교집합
print(a - b)   # {1, 2}           차집합
print(a ^ b)   # {1, 2, 4, 5}     대칭차집합
```
