## `''.join(iterable)`

문자열 리스트(또는 이터러블)를 하나의 문자열로 합치는 함수.

```python
'구분자'.join(iterable)
```

- `iterable` 안의 원소는 **모두 문자열**이어야 함
- 구분자 없이 합칠 때는 `''` 사용

```python
# 기본 사용
words = ['Hello', 'World']
print(' '.join(words))   # Hello World
print(''.join(words))    # HelloWorld
print(', '.join(words))  # Hello, World

# 리스트 → 문자열 변환 (코테 자주 사용)
chars = ['a', 'b', 'c', 'd']
print(''.join(chars))    # abcd

# 숫자 리스트는 직접 불가 → map으로 변환 후 사용
nums = [1, 2, 3]
print(''.join(map(str, nums)))  # 123
```

## `set` / 부분집합 함수

`set`은 **중복 없는 순서 없는** 자료구조.

```python
a = {1, 2, 3}
b = {1, 2, 3, 4, 5}
```

### 주요 부분집합 연산

| 연산 | 의미 |
|------|------|
| `a.issubset(b)` | a가 b의 부분집합인지 (a ⊆ b) |
| `a.issuperset(b)` | a가 b의 상위집합인지 (a ⊇ b) |
| `a <= b` | `issubset`과 동일 |
| `a < b` | **진부분집합** (a ⊂ b, a ≠ b) |
| `a >= b` | `issuperset`과 동일 |

```python
a = {1, 2, 3}
b = {1, 2, 3, 4, 5}

print(a.issubset(b))    # True  → a ⊆ b
print(a.issuperset(b))  # False → a ⊉ b
print(a <= b)           # True
print(a < b)            # True  → 진부분집합 (a ≠ b)
print(a < a)            # False → 같으면 진부분집합 아님
```

### 집합 연산 정리

```python
a = {1, 2, 3}
b = {3, 4, 5}

print(a | b)   # {1, 2, 3, 4, 5}  합집합
print(a & b)   # {3}              교집합
print(a - b)   # {1, 2}           차집합
print(a ^ b)   # {1, 2, 4, 5}     대칭차집합
```

## 파이썬 산술 연산자

### 기본 연산자

| 연산자 | 의미 | 예시 |
|--------|------|------|
| `+` | 덧셈 | `3 + 2` → `5` |
| `-` | 뺄셈 | `3 - 2` → `1` |
| `*` | 곱셈 | `3 * 2` → `6` |
| `/` | 나눗셈 (float) | `7 / 2` → `3.5` |
| `//` | 정수 나눗셈 (내림) | `7 // 2` → `3` |
| `%` | 나머지 | `7 % 2` → `1` |
| `**` | 거듭제곱 | `2 ** 3` → `8` |

### `//` 정수 나눗셈 (Floor Division)

소수점 버리고 **정수만** 반환. 음수에서는 더 작은 쪽으로 내림.

```python
print(7 // 2)    # 3
print(-7 // 2)   # -4  (← -3.5에서 내림)
print(7 // -2)   # -4
```

### `%` 나머지 (Modulo)

```python
print(7 % 3)    # 1
print(10 % 2)   # 0  → 짝수 판별
print(10 % 3)   # 1

# 홀짝 판별 (코테 자주 사용)
n = 7
print("홀수" if n % 2 else "짝수")  # 홀수
```

### `divmod(a, b)`

몫과 나머지를 **동시에** 반환.

```python
q, r = divmod(7, 2)
print(q, r)  # 3 1
```

### 주의: `/` vs `//`

```python
print(6 / 2)    # 3.0  (float)
print(6 // 2)   # 3    (int)

# 코테에서 정수 결과 필요하면 // 사용
# 또는 int(6 / 2)
```
