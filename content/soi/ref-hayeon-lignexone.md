## 1. 지원 분야 관련 직무 역량 및 성장 Vision

**[현장과 연구로 쌓은 강건한 비전]**

학부 시절 네이버 이미지 비전팀 인턴으로 합류해 Face Landmark, Face Detection 모델을 개발했습니다. Face Landmark에서는 회사에서 제공한 이마 데이터를 기존 얼굴 랜드마크 데이터와 통합하는 것이 핵심 과제였습니다. 턱선 끝 랜드마크와 이마 시작 부분이 부드럽게 연결되도록 추가 보정 방식을 적용해 두 데이터를 매끄럽게 병합했습니다. Face Detection에서는 빠르게 움직이는 환경에서 탐지 성능이 저하되는 문제를 분석하고, 프레임 간 IoU 연산 기반 필터링 모듈을 개발해 움직임 변화에 강건한 탐지 안정성을 확보했습니다.

컴퓨터비전 수업 파이널 프로젝트는 YOLOv7 기반 인물 탐지 성능을 최대한 끌어올리는 것이 목표였습니다. 부분적으로 잘린 사람과 갑자기 사라지는 사람을 안정적으로 인식하는 것이 가장 큰 과제였습니다. 모자이크 데이터 증강으로 다양한 형태의 사람에 대한 탐지 능력을 향상시켰고, 후처리 단계에서는 기본 NMS 대신 Soft-NMS를 적용해 갑작스러운 움직임 변화에 따른 불안정한 중복 박스를 효과적으로 제거했습니다. 프레임 간 IoU 필터링으로 시간적 연속성을 추가로 확보하고 하이퍼파라미터를 세밀하게 조정한 결과, 수업 리더보드 1위를 기록했습니다.

BMVC 2025 논문 B-RIGHT에서 사람-물체 상호작용(HOI) 인식의 클래스 불균형 문제를 해결하기 위해 생성 모델로 데이터를 보완하고, VLM, LLM 기반 품질 필터링 파이프라인 전반을 담당했습니다. 생성된 데이터의 품질 보증을 위해 10개 이상의 모델 조합으로 필터링 결과를 비교했습니다. 모델이 신뢰하고 학습할 수 있는 고품질 데이터를 만드는 데 집중한 결과, BMVC 2025 논문 게재로 이어졌습니다.

이러한 경험들을 바탕으로 LIG넥스원 광정보융합연구소에서 강건한 영상 객체 탐지/추적 알고리즘 개발에 즉시 기여하고, 나아가 영상과 다양한 센서 데이터를 융합하는 멀티모달 비전 연구로 전문성을 확장해 나가겠습니다.

## 2. 새로운 시각으로 문제를 포기하지 않고 해결했던 경험

**[낯섦을 기회로 — 제약 속에서 찾은 새로운 시각]**

토론토대학교 파견 중 캐나다 현지 중고 플랫폼 Kijiji와 AI 협업 프로젝트를 수행했습니다. 영상과 이미지만 다뤄온 저에게 텍스트 기반 커머스 데이터는 완전히 낯선 영역이었고, 처음 마주한 문제는 생각보다 단단했습니다. 광고 품질 측정은 VLM 기반 이미지 분석이 정확도 면에서 이상적이었지만 매일 데이터가 업데이트되는 산업 특성상 실시간 적용이 불가능했습니다. 기존 방식을 고집하기보다 문제를 텍스트 기반으로 재정의했습니다. TF-IDF로 광고 텍스트의 핵심 정보를 추출하고, 사용자 행동 데이터 EDA를 통해 '관심 하트' 지표가 광고 품질과 가장 높은 상관관계를 보인다는 인사이트를 도출했습니다. LLM-as-a-judge 방식으로 품질 레이블을 생성한 뒤 경량 selective regressor를 학습시켜, 현업에서 바로 사용 가능한 모델을 완성했습니다.

현재 연구하는 AVEditBench에서도 같은 방식으로 문제에 접근했습니다. 오디오-비디오 편집 평가 분야는 선행 연구가 전무했고, 기존 벤치마크가 크로스모달 보존을 체계적으로 측정하지 못한다는 구조적 한계가 있었습니다. Audio-only, Video-only, AV-Coupled 3개 카테고리 taxonomy를 직접 설계하고, CLAP·ImageBind·ROI 분석을 결합한 다단계 필터링 파이프라인을 구축했습니다.

두 경험은 낯섦과 제약이 오히려 새로운 시각을 열어주었습니다. 주어진 틀 안에서 풀 수 없을 때 문제 자체를 재정의하는 것, 그것이 저의 혁신 방식입니다.

## 3. 타인의 협력을 이끌어 내거나 주도적으로 도와 공동 목표를 달성한 경험

**[무식한 성실함과 먼저 나서는 추진력]**

네이버 인턴 시절, 인턴 2명을 포함한 소규모 연구팀에서 ICCV 2023 HairNeRF 논문의 평가용 데이터 구축을 담당했습니다. VoxCeleb2 데이터셋은 배경과 잡음이 섞인 날 것의 영상이었고, 주어진 얼굴 탐지 모델은 아무리 하이퍼파라미터를 조정해도 엉뚱한 영역을 얼굴로 인식했습니다. 모델 교체 권한도, 전문 어노테이터를 쓸 여건도 없었습니다. 저는 6,000여 장의 이미지를 직접 수작업으로 처리하며 얼굴 외 영역에 검은 박스를 그려 넣어 탐지기가 제 성능을 발휘할 수 있도록 입력 환경 자체를 바꿨습니다. 며칠 밤을 새워 작업을 완성했고, "이 작업 덕분에 논문이 될 수 있었다"는 말씀과 함께 첫 논문을 얻을 수 있었습니다.

학부 시절 Smilegate Membership AI 2기에서는 석사들 사이에서 리더를 맡아 실시간 얼굴 편집 서비스 개발을 이끌었습니다. 대형 모델을 고집하던 팀원들에게 멘토님을 논의에 초대해 실서비스에서 응답 속도와 안정성의 중요성을 함께 들으며 방향을 맞췄습니다. 이후 MobileFaceSwap 경량 모델 도입, Docker 기반 환경 구축, FastAPI 서빙 파이프라인 설계를 직접 담당했습니다. 결국 사용자가 이미지에서 원하는 영역을 직접 지정하면 실시간으로 편집 결과를 확인할 수 있는 서비스를 완성된 형태로 발표할 수 있었습니다.
